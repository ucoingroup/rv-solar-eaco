## 🛠️ 技术组 (4人)

### 1. 技术负责人 (1人) - Tech Lead

**职责**：
- 系统架构设计和技术决策
- 代码审核和质量把控
- 协调技术组内部工作
- 与其他组沟通技术需求
- 进度管理和风险控制

**技能要求**：
- ✅ 精通 Python / JavaScript / TypeScript
- ✅ 有区块链开发经验（Solana优先）
- ✅ 熟悉 API 开发和部署
- ✅ 有项目管理经验

**每日工作**：
- 晨会（30分钟）：同步进度
- 代码审核（2小时）
- 架构设计/决策（2小时）
- 跨组协调（2小时）

**候选人画像**：
- 区块链开发者
- 全栈工程师
- 技术项目经理

---

### 2. 后端开发工程师 (1人) - Backend Developer

**职责**：
- Solana RPC 客户端开发
- 智能合约交互
- 数据库设计和维护
- API 接口开发
- 监控逻辑实现

**技能要求**：
- ✅ Python 熟练
- ✅ Solana/Web3 开发经验
- ✅ API 设计经验
- ✅ 数据库（PostgreSQL/MongoDB）

**技术栈**：
```python

- Python 3.10+
- Solana Web3.py
- FastAPI / Flask
- PostgreSQL / MongoDB
- Docker
```

**每日工作**：
- 编码实现（6小时）

- 代码提交（1小时）

**候选人画像**：
- Python 后端开发者

## 📊 团队协作流程

### 每日站会 (Daily Standup)

**时间**：每天 09:00 (UTC+8)
**时长**：30分钟
**形式**：Telegram 群组语音或文字

**模板**：
```
📅 Day X - 每日站会

1️⃣ 昨日完成：
   - [任务1]
   - [任务2]

2️⃣ 今日计划：
   - [任务1]
   - [任务2]

3️⃣ 阻碍问题：
   - [问题1]
   - [问题2]

4️⃣ 需要支持：
   - [支持需求]
```

---

### 周会议 (Weekly Sync)

**时间**：每周三 20:00 (UTC+8)
**时长**：60分钟
**形式**：Zoom / Google Meet

**议程**：
1. 进度回顾 (15分钟)
2. 问题讨论 (15分钟)
3. 下周计划 (15分钟)
4. 团队反馈 (15分钟)
