## 功能定位

## 核心能力

### 意图识别引擎
```
用户消息 → 分类 → 路由
├── 价格咨询 → 价格表+促销信息
├── 参数规格 → 产品规格表+对比表
├── 定制需求 → 定制问卷+报价请求
├── 物流配送 → 物流时效+费用计算
├── 售后保障 → 退换政策+维修服务
└── 批量采购 → 最低报价+合同模板
```

### 知识库覆盖
- 重庆/上海仓库库存实时查询
- 材质百科: 羊毛/尼龙/丙纶/混纺
- 场景百科: 办公室/酒店/家庭/工程
- 价格词典: 批发价/零售价/工程价
