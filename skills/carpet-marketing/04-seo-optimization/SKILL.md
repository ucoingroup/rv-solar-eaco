# Skill 04: 百度+Google SEO智能优化

## 功能定位
针对"重庆地毯"、"上海地毯"等关键词自动优化官网+电商页面，提升百度/Google双引擎排名。

## 核心能力

### 关键词矩阵
```
核心词: 重庆地毯、上海地毯、地毯厂家
长尾词: 重庆办公室地毯价格、上海酒店地毯批发、重庆方块地毯厂家
问答词: 地毯怎么选、重庆地毯哪家好、上海地毯哪里便宜
```

### 技术SEO
- 结构化数据: Schema.org产品标记(价格/库存/评分)
- 页面速度优化: 图片压缩、WebP格式、自动懒加载
- 移动端适配: 响应式设计+移动优先索引
- 内链策略: 产品页→分类页→博客→FAQ

### 内容SEO
- 每月自动生成10篇SEO文章
- 区域词布局: "重庆南岸区地毯" "上海静安区地毯"
- 问答型内容: 覆盖百度知道/知乎/小红书问答
- 外链建设: 装修平台/建材B2B/本地生活站

### 监控报告
- 关键词排名周报
- 竞争对手对比分析
- 流量来源诊断

## 落地页模板
针对"重庆地毯批发"等高转化词，生成专属落地页:
- 首屏: 产品图+核心卖点+立即询价
- 中段: 工程案例+客户评价+对比表
- 尾端: 联系表单+QQ微信+地址地图
