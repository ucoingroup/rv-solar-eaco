# Skill 07: 跨境价格智能情报

## 功能定位
整合全球API实时汇率(USD/CNY/EUR/AED)，自动计算地毯出口最优报价，覆盖eCNH结算。

## 核心能力

### 汇率监控
```
数据源: 中国银行API / 阿里云汇率服务
监控货币: USD/CNY, EUR/CNY, AED/CNY, GBP/CNY
告警阈值: 汇率波动超过0.5% → 推送通知
```

### 智能报价系统
- 成本加成定价(原材料+加工费+物流+利润)
- 竞争对手对标定价
- 阶梯报价(量少/中等/批发)
- 汇率锁定报价(24小时有效)

### eCNH结算方案
- 了解离岸人民币汇率优势
- 跨境支付最优路径推荐
- 规避汇率风险建议

### 出口文件自动化
- 商业发票(中英文)
- 装箱单
- 原产地证
- 报价单(PDF+Excel)

## 适用场景
- 海外客户询价 → 自动生成含汇率的报价单
- 展会现场报价 → 实时汇率计算
- 长期合同定价 → 锁汇建议
