# Skill 05: 竞品市场智能分析

## 功能定位
实时监控地毯行业价格、竞品动态、市场趋势，输出重庆+上海地区可执行的市场报告。

## 核心能力

### 数据来源
- 1688/淘宝/京东竞品价格抓取
- 百度指数/微信指数区域热度
- 海关出口数据(跨境参考)
- 展会信息(广交会/建博会)

### 分析维度
| 分析项 | 覆盖范围 |
|--------|---------|
| 价格监测 | 同类产品价格带分布、促销节奏 |
| 产品趋势 | 新材质、新工艺、新花色流行度 |
| 区域热度 | 重庆/上海搜索量对比、季节性波动 |
| 客户画像 | 搜索关键词、年龄段、消费能力 |
| 竞争情报 | 头部商家营销策略、内容风格 |

### 输出报告
```
📊 周度市场报告包含:
- 本周热搜词TOP20
- 价格区间分布图
- 竞品爆款分析
- 下周营销建议
```

### 预警机制
- 竞品降价超过10% → 推送告警
- 行业新品出现 → 快速跟进
- 政策变化(贸易/建材) → 影响评估

## 使用场景
- 新品定价策略
- 促销时机选择
- 产品线扩展决策
