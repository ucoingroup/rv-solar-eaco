# Skill 09: 库存智能预警与补货

## 功能定位
连接1688/仓库系统，实时监控地毯库存，自动补货建议，支持重庆+上海双仓调配。

## 核心能力

### 库存监控
- 实时库存可视化(分类+规格+颜色)
- 库存周转天数分析
- 滞销品预警(90天无动销)
- 畅销品预警(库存低于7天销售)

### 智能补货
```
触发条件 → 自动计算 → 推荐方案
库存<安全库存 → 计算补货量+物流时效 → 微信推送采购建议
双十一前 → 提前30天预警 → 备货方案
```

### 双仓调配
- 重庆仓缺货 → 查询上海仓是否有货 → 调拨建议
- 跨区调拨成本计算
- 就近发货推荐

### 数据分析
- 爆款预测(基于历史+季节+营销计划)
- 安全库存计算(基于销量波动)
- ABC分类(按销售额/利润贡献)

## 库存报告
```
📦 周库存状态:
- 当前库存量TOP20产品
- 需要补货产品清单
- 即将过期/淘汰产品
- 建议促销清仓产品
```

## 集成电商后台
- 1688商品库存同步
- 淘宝/京东库存联动
- 微店/小程序库存管理
