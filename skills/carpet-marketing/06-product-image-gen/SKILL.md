# Skill 06: AI地毯效果图生成

## 功能定位
输入产品参数，自动生成电商主图/场景图/详情页图，支持中英文包装。

## 核心能力

### 主图生成
- 纯色背景主图(白底/透明)
- 场景效果图(办公室/酒店/家庭)
- 拼图展示(多角度+细节)
- A+内容图(对比图+卖点图)

### 场景定制
```
酒店大堂 → 地毯+装修氛围渲染
办公室 → 地毯+办公桌椅场景
家庭客厅 → 地毯+家具搭配
儿童房 → 安全地毯+色彩活泼
```

### 多语言包装
- 产品标签: 中英文双语
- 尺寸标注: cm/英寸双制式
- 洗涤说明: 图标+多语言文字
- 品牌水印: © qq12762680 | 重庆上海地毯

### 输出规格
| 用途 | 尺寸 | 格式 |
|------|------|------|
| 电商主图 | 800×800px | PNG/JPG |
| 详情页 | 750×高度不限 | JPG |
| 社交媒体 | 1:1/4:5/16:9 | 多尺寸 |
| 印刷物料 | 300DPI | AI/CDR源文件 |

### 风格选项
- 现代简约 | 欧式古典 | 中东风格 | 北欧风 | 日式和风
