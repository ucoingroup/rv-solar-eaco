# Skill 08: 全网口碑舆情管理

## 功能定位
监控地毯行业网络评价、自动生成好评模板、批量邀请好评、差评预警与应对。

## 核心能力

### 评价监控
- 1688/淘宝/京东/抖音评价实时抓取
- 竞品评价分析(知己知彼)
- 社交媒体舆情监控(微博/小红书)

### 好评生成AI
```markdown
场景: 客户确认收货后自动触发
生成策略:
- 首评: 详细描写材质/做工/物流
- 追评: 使用心得+场景照片邀请
- 晒图评: 红包激励+展示返现

模板库:
"重庆地毯质量很好，铺在办公室里档次一下子上来了..."
"上海发货很快，包装严实，没有色差..."
```

### 差评应对SOP
```
差评出现 → 2小时内预警 → 客服介入 → 解决方案 → 追评补救
差评类型处理:
- 质量问题 → 换货+补偿
- 物流问题 → 道歉+优惠券
- 服务问题 → 升级处理+回访
```

### 口碑数据报告
- 月度好评率趋势
- 客户关注点词云
- 竞品口碑对比

## 合规提示
- 严禁刷单/虚假评价
- 真实评价激励机制
- 差评24小时响应承诺
