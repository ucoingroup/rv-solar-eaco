# Skill 01: 多语言地毯营销内容工厂

## 功能定位
自动生成覆盖中文/英文/阿拉伯语/西班牙语/俄语的地毯产品营销内容，面向重庆+上海本地市场及出口。

## 核心能力

### 1. 产品描述生成
- 输入: 地毯材质、尺寸、图案风格、价格区间
- 输出: 差异化卖点文案（电商平台适配版、社交媒体版、B2B专业版）
- 支持: 京东/淘宝/1688/亚马逊/Alibaba各平台SEO优化

### 2. 标题优化
- 核心关键词: "重庆地毯"、"上海地毯"、"地毯厂家直销"、"批发"
- 长尾词挖掘: 区域+材质+场景组合
- A/B变体: 生成10+标题版本供测试

### 3. 多语言适配
```
中文 → 英文(SEO) → 阿拉伯语(中东市场) → 俄语(东欧) → 西班牙语(拉美)
```

## 营销场景模板

```markdown
场景1: 重庆本地SEO
关键词: "重庆地毯厂家" "重庆办公室地毯" "重庆方块地毯"
场景2: 上海高端市场
关键词: "上海进口地毯" "上海羊毛地毯" "上海酒店地毯"
场景3: 跨境出口
关键词: "China carpet wholesale" "Shanghai rug manufacturer"
```

## 输出格式
- 产品标题 (含emoji)
- 5点USPbullet列表
- 150字产品描述
- Instagram/TikTok短视频脚本(30秒)
- WhatsApp/WeChat群发文案
