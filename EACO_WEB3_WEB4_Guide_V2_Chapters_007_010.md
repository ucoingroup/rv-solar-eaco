## 第8章：10个最常见的入门心理障碍

### 8.1 障碍一："我怕被骗"

**现实**：确实有很多骗局，Web3 骗子比 Web2 更多。

**但**：只要遵循基本规则，被骗概率极低。

**破解方法：**

```
规则一：只用正规渠道
  → 钱包：phantom.app（官方）
  → 交易：jup.ag（官方）
  → 不要从 Google 广告链接进入

规则二：不相信"内部消息"
  → 没有"内幕名额"
  → 没有"私募机会"
  → 没有"保赚项目"

规则三：不分享助记词
  → 永远不告诉任何人
  → 官方永远不会要你的助记词

规则四：先用 $10 测试
  → 不要一开始就投入大额
  → 用小钱走完流程
  → 熟悉后再增加
```

**安全检查清单：**

```
□ 钱包从官网下载
□ 助记词手写，不截图
□ 不点陌生链接
□ 不相信"客服"私聊
□ 先用小额测试
```

### 8.2 障碍二："我不懂技术"

**现实**：Web3 大多数操作不需要写代码。

**类比**：
```
你会用微信，但不需要懂 TCP/IP
你会开车，但不需要懂发动机原理
你会用 Web3，但不需要懂智能合约
```

**Web3 操作的难度：**

| 操作 | 技术要求 | 难度 |
|------|---------|------|
| 创建钱包 | 无 | ⭐（点点点）|
| 买入代币 | 无 | ⭐（点点点）|
| 转账 | 无 | ⭐（点点点）|
| 质押 | 无 | ⭐⭐（多几步）|
| 提供 LP | 无 | ⭐⭐（理解概念）|
| 部署合约 | 有 | ⭐⭐⭐⭐⭐（需要代码）|

**你需要的技术水平：**

```
普通用户：
  → 会用钱包（像用微信一样）
  → 会在 DEX 交易（像淘宝下单一样）
  → 会参与社区（像微信群聊一样）

不需要：
  → 不需要写代码
  → 不需要懂区块链原理
  → 不需要懂加密算法
```

